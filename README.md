# CTI Enrichment Process using LLM

## Description
Ce projet permet l’analyse et l’enrichissement des IOC ThreatFox à l’aide d’un modèle LLM (Mistral-7B via OpenRouter).

## Technologies utilisées
- Python
- OpenRouter API
- JSON
- ThreatFox dataset

## Objectif
Détecter les IOC obsolètes en combinant analyse LLM et règles temporelles.

## Résultat
Génération d’un fichier threatfox_enriched.json avec classification et pourcentage d’obsolescence.
